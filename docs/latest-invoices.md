# Процесс: Просмотр последних инвойсов (Latest Invoices Admin Interface)

Этот процесс позволяет администраторам быстро просматривать хронологический список последних начислений, контролировать их статус оплаты и скачивать прикрепленные PDF-инвойсы.

## Последовательность шагов

### 1. Запрос списка
* Администратор нажимает текстовую кнопку меню `"Последние инвойсы"` или вводит команду `/latest_invoices`.
* Telegram-бот отправляет запрос `get_accruals_paginated` с параметрами пагинации (`skip`, `take`) в сервис Accountant.

### 2. Обработка данных (Accountant)
* Accountant запрашивает последние начисления из БД, упорядоченные по `periodId desc` и `firstSeenAt desc`.
* Каждое начисление сопоставляется с таблицей `Invoice` (по `accountExternalId` и `periodId`), чтобы определить доступность PDF-файла в S3.
* Данные сериализуются (включая исходный JSON-объект начисления `rawJson`) и возвращаются в бот.

### 3. Отображение в боте
* Бот форматирует полученные данные:
  * **Сумма**: извлекается точное числовое значение из `rawJson.accrual` (выбирается первое непустое поле по приоритету: `accruedAmount` -> `amountToPay` -> `paidAmount`), вместо склеенного технического текста разметки.
  * **Статус**: технические свойства (кнопки оплаты, флаги задолженности) преобразуются в читаемый русский статус (например, `Оплачено` или `Ожидает оплаты`).
  * **Дата регистрации**: выводится дата `firstSeenAt` (когда начисление впервые обнаружено парсером), приведенная к часовому поясу из конфигурации.
* Под текстом формируется inline-клавиатура:
  * Кнопки скачивания доступных инвойсов располагаются в столбик, подписанные порядковым номером строки начисления (например, `📄 Скачать инвойс #1`, `📄 Скачать инвойс #2`).
  * Под кнопками скачивания выводится горизонтальный ряд навигации: `[⬅️ Назад]` и `[➡️ Вперед]` (если страниц больше одной).

### 4. Интерактивная навигация и скачивание
* Клик по навигации (`admin_latest_accruals_(\d+)`) перерисовывает сообщение (через `editMessageText`) с данными новой страницы.
* Клик по кнопке скачивания отправляет событие загрузки PDF из S3. Бот присылает файл напрямую в чат.
